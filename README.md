# Ecommerce Intership Project

This is my project for internship. It is a shopping website.


**Main function:**

- Log in
- View products list
- View single product
- Add to cart
- Place an order
- Admin page: view order

## Technologies
- **React:** Front-end
- **Meteor:** Full-stack
- **MongoDB:** Database

## Set-up Project
1. Clone source code from branch `master`
2. Run `meteor`
